# GEO/SEO Console — 设计系统

## 🎨 设计理念

**深空奢华风** — 极度克制的奢华，深邃黑底 + 米白文字 + 单一玫瑰金高光

```
灵感来源：高级腕表、航空仪表盘、豪华汽车内饰
关键词：克制、精准、高端、专业
```

## 🧩 组件规范

### 按钮
- **Primary**: 玫瑰金背景，深色文字
- **Secondary**: 透明背景，金色边框
- **Ghost**: 完全透明，金色 hover

### 卡片
- 深色背景 `#0F0F0F`
- 顶部 1px 金色渐变线
- 6px 圆角

### 输入框
- 深色背景，微妙边框
- 聚焦时金色边框发光
- 16px 内边距
